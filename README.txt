-------------------------------------------------------
 README - Sistema de Gestão de Campeonatos de BASEBALL
-------------------------------------------------------

Quando o programa é executado pela primeira vez,
como não existem ficheiros para leitura,
são acrescentados dados por defeito.

Se não houver utilizadores é acrescentado
um administrador por defeito com login: 1 e senha: admin.
Pode sempre entrar como visitante inserindo o login 0.
